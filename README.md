# streamlit_practise
streamlit浅尝


### 项目准备

* 创建虚拟环境
```shell
python3 -m venv venv_streamlit
```
* 激活虚拟环境
```shell
source venv_streamlit/bin/activate
```
* 将依赖写入requirements.txt
```shell
pip3 freeze > requirements.txt

# 仅追加
pip3 freeze >> requirements.txt
```
